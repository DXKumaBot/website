title: 星星（货币）相关

:::note[注意]
以下指令仅普通版可用
:::

## 查询星星数量

查询用户星星的具体数量

```regexp frame="none"
查询星星|星星查询|查星星|我有多少星星|我有多少星
```

<details>
<summary>示例</summary>
```regexp frame="none"
查星星
```
</details>

## 查询星星明细

查询用户星星的收支情况

```regexp frame="none"
查星星明细|星星明细
```

<details>
<summary>示例</summary>
```regexp frame="none"
星星明细
```
</details>
