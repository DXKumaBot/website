## 按等级筛选

生成使用等级进行筛选的成绩列表

<Tabs syncKey="bot-type">
  <TabItem label="官B版">
    ```regexp frame="none"
    @迪拉熊 /成绩表 等级 <level:number>+?
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    @迪拉熊 /成绩表 等级 14+
    ```
    </details>
  </TabItem>
  <TabItem label="普通版">
    ```regexp frame="none"
    (list|成绩表|成绩列表|成绩表格) <level:number>+?
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    list 14+
    ```
    </details>
  </TabItem>
</Tabs>

| 参数    | 简介 | 类型 | 范围 | 重要度 | 示例 |
| ------- | ---- | ---- | ---- | ------ | ---- |
| `level` | 等级 | 整数 | 1~15 | 必填   | `14` |

## 按DX星数筛选

<Tabs syncKey="bot-type">
  <TabItem label="官B版">
    ```regexp frame="none"
    @迪拉熊 /成绩表 dx星 <value:number+>
    ```
    <details>
    <summary>示例</summary>

    </details>
  </TabItem>
  <TabItem label="普通版">
    ```regexp frame="none"
    xx(list|表|列表|表格) <value:number+>
    ```
    <details>
    <summary>示例</summary>

    ```regexp frame="none"
    xxlist 3 4 5
    ```
    </details>
  </TabItem>
</Tabs>

| 参数    | 简介       | 类型         | 范围 | 重要度 | 示例    |
| ------- | ---------- | ------------ | ---- | ------ | ------- |
| `value` | DX星星数量 | 单或多个整数 | 0~5  | 必填   | `3 4 5` |
