## 标准

生成标准评分对象曲目列表（B50）

:::tip[你知道吗？]
这里的标准并非指代标准乐曲
:::

<Tabs syncKey="bot-type">
  <TabItem label="官B版">
    ```regexp frame="none"
    @迪拉熊 /(评分对象曲目|b50) 标准
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    @迪拉熊 /b50 标准
    ```
    </details>
  </TabItem>
  <TabItem label="普通版">
    ```regexp frame="none"
    (迪拉熊|dlx)(b50|50)
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    dlx50
    ```
    </details>
  </TabItem>
</Tabs>

## 偷师

生成由双方都存在成绩的谱面组成的评分对象曲目列表

:::tip[排序规则]
先选出"被查询者"分数比"查询者"最低B35/B15分还高的曲目，然后按照"被查询者分数"减去"查询者分数"的差值，从大到小排列。
:::

:::note[注意]
该指令仅普通版可用
:::

:::note[注意]
该指令要求在命令中附带`@某人`
:::

```regexp frame="none"
(迪拉熊|dlx)cf(50)?
```

<details>
<summary>示例</summary>
```regexp frame="none"
dlxcf
```
</details>

## 按寸止筛选

生成使用寸止进行筛选的评分对象曲目列表

:::tip[你知道吗？]
寸止：成绩刚好卡在下个评级线附近，差一点点就能升段，范围是"下个评级的最低达成率"减去（"绝赞50落的101-达成率"和"TAP粉的101-达成率"里较小的那个的绝对值）到"下个评级的最低达成率"之间。
:::

<Tabs syncKey="bot-type">
  <TabItem label="官B版">
    ```regexp frame="none"
    @迪拉熊 /(评分对象曲目|b50) 寸止
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    @迪拉熊 /b50 寸止
    ```
    </details>
  </TabItem>
  <TabItem label="普通版">
    ```regexp frame="none"
    (迪拉熊|dlx)(sunn|sun|cun|寸|🤏)(50)?
    ```
    <details>
