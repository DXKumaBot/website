## 注册机厅

注册并为当前聊天绑定机厅

<Tabs syncKey="bot-type">
  <TabItem label="官B版">
    ```regexp frame="none"
    @迪拉熊 /管理机厅 注册 <name:string>
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    @迪拉熊 /管理机厅 注册 xxx机厅（xxx店）
    ```
    </details>
  </TabItem>
  <TabItem label="普通版">
    ```regexp frame="none"
    注册机厅 <name:string>
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    注册机厅 xxx机厅（xxx店）
    ```
    </details>
  </TabItem>
</Tabs>

| 参数   | 简介     | 类型 | 重要度 | 示例               |
| ------ | -------- | ---- | ------ | ------------------ |
| `name` | 机厅全称 | 文字 | 必填   | `xxx机厅（xxx店）` |

## 查询卡数

查询当前聊天绑定的机厅卡数

<Tabs syncKey="bot-type">
  <TabItem label="官B版">
    ```regexp frame="none"
    @迪拉熊 /查询卡数 [word:string] | @迪拉熊 (机厅|看看|jt|[word:string])?有?(多少人|多少卡|几人|几卡|几|jr|j)
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    @迪拉熊 /查询卡数
    ```
    </details>
  </TabItem>
  <TabItem label="普通版">
    ```regexp frame="none"
    (机厅|看看|jt|[word:string])?有?(多少人|多少卡|几人|几卡|几|jr|j)
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    机厅有多少卡
    ```
    </details>
  </TabItem>
</Tabs>

| 参数   | 简介           | 类型 | 重要度 | 示例  |
| ------ | -------------- | ---- | ------ | ----- |
| `word` | 机厅全称或别名 | 文字 | 可选   | `xxx` |

## 修改卡数

更新指定机厅卡数

<Tabs syncKey="bot-type">
  <TabItem label="官B版">
    ```regexp frame="none"
    @迪拉熊 /修改卡数 <word:string> (增加|减少|设为|add|reduce|set|加|减|为|＋|－|＝|+|-|=) <num:number> | @迪拉熊 <word:string> (加|减|为|＋|－|＝|+|-|=)? <num:number> (人|卡)?
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    @迪拉熊 /修改卡数 xxx 设为 0
    ```
    </details>
  </TabItem>
  <TabItem label="普通版">
    ```regexp frame="none"
    <word:string> (加|减|为|＋|－|＝|+|-|=)? <num:number> (人|卡)?
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    xxx 0
    ```
    </details>
  </TabItem>
</Tabs>

| 参数   | 简介           | 类型 | 重要度 | 示例  |
| ------ | -------------- | ---- | ------ | ----- |
| `word` | 机厅全称或别名 | 文字 | 必填   | `xxx` |
| `num`  | 卡数           | 整数 | 必填   | `0`   |
