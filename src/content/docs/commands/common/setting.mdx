## 设置收藏品

为水鱼源设置收藏品

### 姓名框

<Tabs syncKey="bot-type">
  <TabItem label="官B版">
    ```regexp frame="none"
    @迪拉熊 /设置收藏品 姓名框 <id:number>
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    @迪拉熊 /设置收藏品 姓名框 458001
    ```
    </details>
  </TabItem>
  <TabItem label="普通版">
    ```regexp frame="none"
    ((set|s)plate|(设置|设)姓名框) <id:number>
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    splate 458001
    ```
    </details>
  </TabItem>
</Tabs>

## 绑定账号

绑定其他平台的账号

### 落雪

:::tip[你知道吗？]
个人API密钥可以在[账号详情](https://maimai.lxns.net/user/profile?tab=thirdparty)页获取
:::

<Tabs syncKey="bot-type">
  <TabItem label="官B版">
    ```regexp frame="none"
    @迪拉熊 /绑定 落雪 <token:string>
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    @迪拉熊 /绑定 落雪 ZlxbZW-w3k2EpABLey8evn8G4TevG3wE2T4D4YCzRPX=
    ```
    </details>
  </TabItem>
  <TabItem label="普通版">
    ```regexp frame="none"
    (绑定|bind) 落雪 <token:string>
    ```
    <details>
    <summary>示例</summary>
    ```regexp frame="none"
    绑定 落雪 ZlxbZW-w3k2EpABLey8evn8G4TevG3wE2T4D4YCzRPX=
    ```
    </details>
  </TabItem>
</Tabs>

| 参数    | 简介            | 类型 | 位数 | 重要度 | 示例                                           |
| ------- | --------------- | ---- | ---- | ------ | ---------------------------------------------- |
| `token` | 落雪个人API密钥 | 文字 | 44位 | 必填   | `ZlxbZW-w3k2EpABLey8evn8G4TevG3wE2T4D4YCzRPX=` |

### QQ

:::note[注意]
该指令仅官B版可用
:::

```regexp frame="none"
@迪拉熊 /绑定 qq <qq:number>
```

<details>
<summary>示例</summary>
```regexp frame="none"
@迪拉熊 /绑定 qq 10000
```
</details>

| 参数 | 简介 | 类型 | 位数   | 重要度 | 示例    |
| ---- | ---- | ---- | ------ | ------ | ------- |
| `qq` | QQ号 | 整数 | 5~10位 | 必填   | `10000` |
